Preguntas

   a) Si.
   b) No.
4. El Ransomware:
   a) Es un tipo de software malicioso o Malware.
   b) Está diseñado para negar acceso a un sistema informático o datos hasta pagar un rescate.
   c) Se propaga a través de correos electrónicos de phishing o visitando un sitio web infectado.
   d) Todas las anteriores.
5. Algunas víctimas de Ransomware pagan para recuperar sus archivos, teniendo garantía de recuperar sus archivos si pagan el rescate.

   b) Falso.

   a) Actualizar software y sistemas operativos con los últimos parches.

   c) Seguir prácticas seguras al navegar por internet.
   d) Todas las anteriores.

   a) Si.
   b) No.

   b) Escanear correos (entrantes/salientes) para detectar amenazas y filtrar archivos ejecutables.
   c) Configurar Cortafuegos (firewalls) para bloquear acceso a direcciones IPS maliciosas.
   d) Todas las anteriores.
